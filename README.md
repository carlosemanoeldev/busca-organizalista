# Lista de Usuarios

## Descrição

 Nesse projeto eu criei uma aplicação que mostra uma lista de usuários
 baseada num json fictício que se encontra dentro da pasta dados. Eu
 utilizei html,css e js para desenvolver. Utilizei de boas práticas que
 aprendi nas aulas com professor Emerson ex: Html semântico, estruturação
 de pastas e etc. Também tentei deixar o código o mais limpo possível e organizado
 com padronização nos espaçamentos a fim de deixar mais claro para quem for 
 analisar o código.

## Funcionalidades 

    1 - Buscar usuários por nome ou email
    2 - Ordena a lista de usuários por ID, idade ou nome em ordem alfabética

## Referente a Lógica

    O código está comentado explicando o que cada função 
    faz. A fim de tirar todas as dúvidas

    
## Caracteristicas Visuais

    Optei por um tema dark um pouco voltado para o azul
    com uma interface minimalista que transmita as informações 
    com clareza.


## Método de Utilização

    1 - Abra pela index.html
